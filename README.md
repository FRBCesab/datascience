## Setup

You'll need these two R packages: `rmarkdown` and `xaringan`.

```{r}
install.packages(c("rmarkdown", "xaringan"))
```


## Contributing

To modify the slides, edit the `index.Rmd` and then:

```{r}
rmarkdown::render("index.Rmd")
```
